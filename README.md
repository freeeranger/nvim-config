# Neovim config
My personal neovim configuration

## Requirements:
* Neovim >= 0.10.0 (compiled with LuaJIT)
* Ripgrep
* Git

## Installation:
Run the following command: `git clone https://github.com/freeeranger/nvim-config.git ~/.config/nvim`
